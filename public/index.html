How does it work

Ce système de validation utilise la blockchain Hedera Hashgraph pour valider les documents, permettant aux utilisateurs d'être certains que leur fichier est la forme authentique et non falsifiée du document. Lorsqu'un document est téléchargé, le système stocke le document et génère un hash Hedera à partir de celui-ci. Ensuite, il utilise les API Hedera pour ancrer ce hash sur la blockchain. À l'avenir, lorsque le même fichier est téléchargé dans la section de validation, le système génère à nouveau un hash et vérifie si ce même hash existe sur la blockchain. Si tel est le cas, le document est valide. Sinon, soit le fichier n'a pas été téléchargé dans le système auparavant, soit le document est invalide.
